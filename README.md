# adesso-input
A Web Component for input.

Feel free to make your "Pull Request" 🏆

  # installation:
  `npm install adesso-input`
  
  # usage:
  `
  <adesso-input  \n
    label="Email"  \n
    type="email"  \n
    animated  \n
    validation-rules="email,required">  \n
      <i slot="rightIcon" className="fas fa-dragon"/> \n
  </adesso-input>
  `
